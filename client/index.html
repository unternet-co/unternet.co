Operator

Think and do with an agent that's yours.

  • Direct manipulation: Chat is cumbersome. Agents should be quick. Operator gives you the best of both: mini apps for fast interactions, and a chat thread for everything else.
  • Workspace-based: We know there's a lot happening in your life. Organize different workspaces with access to different resources, to help keep both you and your agent focused.
  • Open & customizable: Missing something? It's infinitely extensible. Add the data and behaviours you need via the web applets protocol.
  • Local & private: Operator is your own. No one else controls your agent. You can connect it to a cloud-based model if you want, but by default it will run locally through Ollama.
